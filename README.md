# nls_race_live_stream

🏁 NLS Race Control Dashboard

A unified, telemetry-style dashboard for Nürburgring Langstrecken-Serie (NLS) races that combines live streams and race data into a single viewing experience.

📌 Background

Nürburgring endurance racing is one of the most exciting formats in motorsport:

Multiple classes on track simultaneously
Dozens of cars competing across categories
Continuous battles happening throughout the field

However, the viewing experience is fragmented.

The official race is streamed on YouTube
Some teams (e.g., Verstappen Sim Racing) stream onboard footage on Twitch
Additional feeds are scattered across platforms

Unlike Formula 1, there is no unified platform (such as F1 TV) that allows viewers to:

Watch multiple camera angles simultaneously
Follow specific cars
Monitor live timing alongside video
💡 Concept
Unified NLS Race Control Dashboard

This project is a prototype dashboard designed to replicate a race engineer’s view rather than a traditional broadcast.

It consolidates multiple data sources into a single interface for a more immersive and analytical race experience.

🚀 Features
🔴 Multi-Stream Viewing
Twitch onboard feed (e.g., Verstappen Sim Racing)
Official YouTube race broadcast
Parallel viewing for better race context
📊 Live Telemetry-Style Leaderboard
Real-time standings
Class-based filtering (e.g., SP9 / GT3)
Gap tracking between cars
Position change tracking
🎯 Why This Matters

In endurance racing, the most important moments are not always shown on the main broadcast.

Examples:

Key battles may occur outside the top positions
Strategic decisions (pit stops, pace management) often go unnoticed
Team onboard feeds reveal perspectives not available in the official stream

This dashboard enables users to:

Monitor multiple race narratives simultaneously—similar to a real pit wall environment.

🔮 Future Scope

This project is an initial prototype with several potential enhancements:

Driver-focused tracking across multiple feeds
Automatic battle detection and stream highlighting
Stint analysis and pit strategy visualization
Multi-class race segmentation
A full “F1 TV-style” experience tailored for endurance racing
🌐 Vision

Motorsport generates a large volume of data, but current broadcasts expose only a fraction of it.

The future of race viewing is:

Interactive
Multi-angle
Data-driven

This project aims to unify existing fragmented components into a cohesive experience.
